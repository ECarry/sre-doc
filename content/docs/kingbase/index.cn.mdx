# KingbaseES

## 产品简介

金仓数据库管理系统 KingbaseES（简称 KES）是中电科金仓（北京）科技股份有限公
司（简称电科金仓）研发的、具有自主知识产权、获得自主原创资质认证的通用数据库产品。
该产品面向全行业、全客户关键应用的企业级大型通用数据库管理系统，适用于事务处理类
应用、数据分析类应用、海量时序数据采集检索等应用场景，可用作管理信息系统、业务及
生产系统、决策支持系统、多维数据分析、全文检索、地理信息系统、图片搜索、时序数据
等的承载数据库。

## 产品定位

KES 是电科金仓的核心产品，具有大型通用、“三高”（高可靠、高性能、高安全）、“三易”（易管理、易使用、易扩展）、运行稳定等特点。

## 应用场景

KES 主要面向事务处理类、数据分析类、海量时序数据采集检索类应用。广泛应用在联
机事务交易的各业务场景中，支持高并发、简单事务、低响应延迟应用。典型系统如金融行
业的收费系统、结算系统，医疗行业HIS 系统、PACS 系统等。提供大规模并行计算及列存
技术，支持海量数据分析处理，进行数据挖掘和分析，普遍用于数据仓库、决策支持、高级
分析等分析类应用系统。支持连续聚合、时间分区、压缩技术，在能源、金融、物联网、交
通等行业主要用于实时数据监控及采集，典型应用如智能电网、管网、电表、发电设备的集
中监测，实时路况、路口流量监测、卡口数据等。
