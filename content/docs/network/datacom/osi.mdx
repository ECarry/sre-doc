## 网络参考模型与标准协议

### OSI参考模型

|              |                                 |
| ------------ | ------------------------------- |
| 7.应用层     | 提供应用程序间通信              |
| 6.表示层     | 数据格式化、加密解密            |
| 5.会话层     | 建立、维护、终止会话            |
| 4.传输层     | 端到端传输，提供可靠/不可靠传输 |
| 3.网络层     | 路由选择、逻辑地址              |
| 2.数据链路层 | 物理地址、错误检测              |
| 1.物理层     | 比特流传输                      |
